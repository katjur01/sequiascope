Network Graph

The Network Graph module provides an interactive visualization of protein-protein interactions and pathway-level insights using patient-specific gene data. This module leverages Cytoscape.js for dynamic graph rendering and integrates STRING DB interactions with pathway annotations from KEGG DB .

Overview of Features

Interactive Network Visualization

The network graph displays interactions between patient genes and their associated pathways.

  • Nodes: Represent genes, colored by fold-change values.
  • Edges: Represent STRING DB interactions, with adjustable confidence thresholds.
  • Dynamic Layout: Nodes and edges can be repositioned for better visualization.

Tip: Use the drag-and-drop functionality to rearrange nodes for a clearer view of pathway interactions.

There are two types of graphs available:

  1. Main Graph
    This graph visualizes interactions among genes within a selected pathway. You can customize the graph using the following options located above the graph:

    • Pathway: Select a pathway to display interactions among genes within it.
    • Layout Algorithms: Choose from multiple layout algorithms to optimize graph readability:
      • cola: Ideal for hierarchical structures and smaller graphs.
      • FCoSE: Best for large and complex networks.
    • Add Selected: Highlight marked variants or fusions from previous modules (somatic, germline, fusion) in the network.
    • Tissue: Choose a reference tissue to color the network accordingly.

    Node Interaction:

    • Select nodes by clicking on them. Hold the Shift key to select multiple nodes.
    • Drag selected nodes to reposition them within the graph.
    • Unselect nodes by clicking on the same node again or clicking on an empty space in the graph area.

    Graph Controls:

    • Clear Selection: Unselect all selected nodes.
    • Select First Neighbors: Select all nodes directly connected to the currently selected nodes.
    • Fit Graph: Automatically adjust the view to fit all nodes within the visible area.

    For more information about the layout algorithms, refer to the Cytoscape.js Layouts Blog.

  2. Sub Graph
    This graph is displayed when you select nodes from the main graph, allowing you to focus on specific genes and their interactions. Additional features include:

    • Add Genes: Include genes not present in the pathway selected in the main graph. Nodes with a gray color indicate genes not part of the patient’s dataset or unrecognized gene names.
    • Remove Nodes: Exclude nodes from the sub graph as needed.

    Tables Under Graphs:

    • Tables provide detailed information about the nodes present in the graphs, including gene names, fold-change values, p-values, pathway names, and the number of pathways the gene is present in.
    • Sort tables by clicking on column names or search for specific nodes using the search box.
    • By clicking on a row in the table, the corresponding node in the graph will be highlighted as yellow.

Customization Options

The module allows users to customize the network graph:

  • Add/Remove Nodes: Include or exclude genes to refine the network.
  • Confidence Thresholds: Adjust STRING DB confidence levels to filter edges.
  • Filter by Evidence Type: Display edges based on specific evidence types (e.g., experimental, text mining).

Note: Adjusting confidence thresholds may significantly alter the network structure. Use this feature to focus on high-confidence interactions.

Note: When adding genes not present in the original pathway, ensure that the gene names are correctly formatted to avoid unrecognized entries.

Pathway Integration

The module integrates KEGG pathway data to contextualize gene interactions:

  • Pathway Highlighting: Highlight nodes and edges associated with specific pathways.
  • Oncogenic and Pathogenic Genes: Highlight genes flagged in previous modules (e.g., somatic, germline, fusion).

Tip: Use pathway highlighting to identify disrupted interactions within key biological processes.

Tumor-Normal Comparisons

For tumor-normal comparisons, node colors represent the log2 fold-change between tumor and matched healthy control tissues:

  • Upregulated Genes (fold-change > 1): Nodes are colored red, with intensity increasing as the fold-change value increases.
  • No Change (fold-change ≈ 0): Nodes are colored white or light gray.
  • Downregulated Genes (fold-change < -1): Nodes are colored blue, with intensity increasing as the fold-change value becomes more negative.
